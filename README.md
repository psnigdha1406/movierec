# MOVIE RECOMMENDATION SYSTEM
## DESCRIPTION:
This is a movie recommendation system that suggests movies based on user ratings or genres. 
It Implements collaborative filtering or content-based filtering techniques while giving us the suggestions.
## LANGUAGES USED:
The following programming languages are used to prepare the project:

->HTML

->CSS

->JAVASCRIPT
## PROCEDURE:
Step 1: Select a movie from the given options.

Step 2: Enter a USER ID from (0-1)

Step 3: Click 'Get Recommendation' to get few movie recommendations based on your preferences
